Por que traduzir?

    Traduzir um projeto torna ele acessível para pessoas de diferentes países, aumentando sua popularidade e usabilidade.

    Existem vários projetos que hospedam seus pacotes de idioma por lá, para que os interessados ajudem a traduzir para os mais variados idiomas.

    O Deepin é um exemplo de distribução que utiliza o Transifex para tradução, e o projeto também busca colaboradores para traduzir o sistema para o português do Brasil.

    Além disso, o Libre Translate é uma alternativa open source que permite traduções automáticas, garantindo privacidade e personalização. Ele pode ser usado para traduzir texto diretamente ou integrado a outras aplicações através de sua API REST.

    Outra opção é o Devtranslate, que promove uma melhor loclização por meio de um catálogo de traduções e facilita o processo de tradução de conteúdo para outros idiomas.

    Esses projetos abrem portas para uma comunidade colaborativa e acessível, permitindo que mais pessoas contribuam para a tradução de conteúdos de tecnologia em português.


    Transifex

  • O Transifex é um site muito popular para tradução de projetos, normalmente de softwares, de desenvolvimento comunitário.
  • O Transifex (anteriormente conhecido como Indifex) é um sistema de gerenciamento de globalização (GMS), um software proprietário e uma plataforma de tradução baseada na web.
  • Ele visa projetos técnicos com conteúdo atualizado com frequência, como software, documentação e sites, e incentiva a automação do fluxo de trabalho de localização integrando-se a ferramentas comuns de desenvolvedor.
  • O Transifex é um SaaS (software as a Service), com plano pago e gratuito para localização de software de código aberto. O Transifex era originalmente um projeto de código aberto, mas foi descontinuado em 2013.
  • O site fornece uma plataforma de hospedagem para arquivos de tradução e funções de redes sociais, como feeds, fóruns de discussão, sugestões de tradução e votação para permitir que os tradutores colaborem. O Transifex é escrito usando Django e Python pelo fundador Dimitris Glezos em um projeto Google Summer of Code.

    • História

  • O Transifex, que começou como um projeto do Google Summer of Code para resolver um problema do Projeto Fedora, evoluiu para um movimento completo apoiado por uma nova startup, a Transifex. O criador do Transifex, Dimitris Glezos, trabalha desde 2007 para criar a visão Transifex de localização e ferramentas i18n.
  • Em março de 2014, a transifex.com tinha mais de 17.000 traduções de projetos hospedadas em 150 idiomas, incluindo Creative Commons, Coursera, Django e Django-cms, Dolibarr, Eventbrite, Mercurial, Bitbucket, GlobaLeaks, TalentLMS, MeeGo, OpenStack, Pinterest, qBittorrent, Reddit, Xfce, Wheelmap.org, VLC, e outros.
  • A empresa estava anteriormente localizada na Grécia, mas agora está sediada no Vale do Silício. Em 2025, a Transifex foi comprada pela XTM International.

    Deepin

  • O deepin (pronuncia-se "dipen") é uma distribuição Linux baseada no Debian Stable. Apresenta uma interface própria, chamada Deepin Desktop Environment (DDE); além de, uma acervo de aplicações próprias. É desenvolvida pela Wuhan Deepin Technology Company Limited, localizada em Wuhan, China.

    • Visão Geral

    A distribuição ganhou popularidade ao apresentar um elevado cuidado visual com a sua interface e aplicativos próprios. Isso acabou surpreendendo muitos usuários, que olhavam o Linux como um sistema de visual antiquado ("da tela preta", como muitos dizem).

    Um dos maiores destaques do deepin é a Loja de Aplicativos. Além de apresentar um visual incrível e fácil de usar; permite que usuários leigos possam instalar e remover aplicativos com um clique.

    No Brasil, a Comunidade deepin do Brasil (simplesmente "deepin Brasil") originou-se com o objetivo de auxiliar os usuários brasileiros e participar ativamente do desenvolvimento do sistema.


    História

  • O deepin foi originalmente lançado sob o nome Hiwix 0.1 em 28 de fevereiro de 2004. 'Linux' foi acrescentado ao título das versões 0.2 em diante. Esta versão inicial foi baseado em Morphix, o que significa que ele tem uma história mais longa do que a distribuição na qual agora é baseada: Debian. Também já foi usado IceWM para sua interface de usuário.
  • Desde então, a base padrão e interface de ambos foram alterados várias vezes. Em dezembro de 2009 o nome foi mudado para Linux Deepin e a interface padrão para uma versão modificada do GNOME. Isto, aliado com mudança para a base Ubuntu, representou uma significativa mudança de marca para o projeto. Os desenvolvedores continuaram com esta fórmula por vários anos, até mesmo através da atualização para o GNOME 3 e as críticas que o mesmo tinha recebido em seu lançamento.
  • Eventualmente, foi decidido fazer a troca do GNOME para a Deepin Desktop Environment. 'Linux' também foi mais tarde retirado do nome. Desde a versão Deepin 15.1, o projeto mudou a base do sistema para o Debian.
